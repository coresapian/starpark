# =============================================================================
# Installation Instructions
# =============================================================================
# 
# Basic installation (core functionality):
#   pip install numpy shapely geopandas pyproj
#
# Development installation (includes testing and type checking):
#   pip install -r requirements_raycasting.txt
#
# =============================================================================
# Dependency Notes
# =============================================================================
#
# numpy: Core vectorized operations for all geometric calculations
#        - Azimuth/elevation angle computations
#        - ENU coordinate transformations
#        - Obstruction profile generation
#
# shapely: 2D geometric operations
#          - Building footprint polygon operations
#          - Point-in-polygon tests for building containment
#          - Buffer operations for search radius
#
# geopandas: Geospatial data management
#            - Building data loading from shapefiles/GeoJSON
#            - Spatial joins and filtering
#            - GeoJSON export for frontend visualization
#
# pyproj: Coordinate reference system transformations
#         - WGS84 to local ENU conversions
#         - Accurate ellipsoid calculations
#         - Datum transformations
#
# =============================================================================
# Version Compatibility
# =============================================================================
#
# Python: 3.8+
# NumPy: 1.21+ (required for type annotations)
# GeoPandas: 0.10+ (required for GeoDataFrame operations)
# Shapely: 1.8+ (required for vectorized operations)
# PyProj: 3.2+ (required for CRS transformations)
#
# =============================================================================
